## 🎯 Quick Upload Guide:

### Option 1: Upload Everything at Once (Easiest)
1. Select ALL files and folders from root directory
2. Upload to server's `public_html` folder
3. **EXCEPT** - Don't upload the `public_html` folder itself

### Option 2: Upload Folder by Folder
Upload these folders one by one:
1. All `.html` files (select all HTML files)
2. `css/` folder
3. `js/` folder
4. `images/` folder
5. `includes/` folder
6. `events/` folder
7. `fonts/` folder (if exists)
8. `robots.txt` and `sitemap.xml`

---

## ⚠️ IMPORTANT:

**Your website needs ALL these files to work:**
- Without `css/` → Website will look broken (no styling)
- Without `js/` → Navigation and features won't work
- Without `images/` → No logos, photos, or graphics
- Without `includes/` → Navigation and footer won't load
- Without other HTML pages → Links will be broken

**Think of it like a house:**
- `index.html` = Front door
- `css/` = Paint and decoration
- `js/` = Electricity and plumbing
- `images/` = Furniture and decorations
- Other HTML pages = Other rooms

You need ALL of them for the house to work!
